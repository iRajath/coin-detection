<h1> Coin Detection using OpenCV </h1>
<h2> Overview </h2>

This project detects coins in an image using OpenCV image processing techniques and blob detection.
Requirements

    Python
    OpenCV
    NumPy
    Matplotlib

Run the Project

    Install required libraries
    Open the notebook file
    Run all cells step by step
    View the detected coins in the output

<h2> Output </h2>

The program detects coins from the input image and displays the total number of detected coins.
